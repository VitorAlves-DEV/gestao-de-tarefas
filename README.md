# 🗂️ Sistema de Gestão de Tarefas

Projeto desenvolvido como Teste de Performance 1 da disciplina de Python.

## 📋 Descrição

Sistema de gestão de tarefas desenvolvido em Python que permite ao usuário adicionar, listar, marcar como concluída e remover tarefas por meio de um menu interativo no terminal.

## ⚙️ Funcionalidades

- ✅ Adicionar nova tarefa
- 📋 Listar todas as tarefas com status
- ✔️ Marcar tarefa como concluída
- ❌ Remover tarefa
- 🚪 Encerrar o programa

## 🛠️ Tecnologias Utilizadas

- Python 3

## ▶️ Como Executar

1. Clone o repositório ou baixe o arquivo `.py`
2. Execute o arquivo no terminal:

python gestao_tarefas.py

3. Escolha uma opção do menu digitando o número correspondente (1-4) ou `sair` para encerrar
